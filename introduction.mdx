# Infraestrutura de Integração Municipal

A IplanRio desenvolve e mantém a infraestrutura de integração que conecta as secretarias municipais da Prefeitura do Rio de Janeiro, permitindo compartilhamento seguro de dados e serviços entre SMS, SME, SMTR, SMSURB, SMF, SEOP e demais órgãos municipais.

## Casos de Uso por Secretaria

### Integração de Dados entre Secretarias
- **SMS + SME**: Dados de saúde escolar para planejamento de políticas públicas
- **SMF + Todas**: Dados financeiros para orçamento e transparência municipal

### Processamento de Dados Municipais
- **Pipelines ETL**: Censo escolar (SME), dados de saúde (SMS), receita municipal (SMF)
- **Transformações dbt**: Modelos analíticos para tomada de decisão municipal
- **APIs de Consulta**: Busca unificada de informações municipais
