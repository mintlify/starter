## Write and preview content

<AccordionGroup>
  <Accordion title="Use the web editor">
    Open the [Mintlify dashboard](https://app.mintlify.com) and work directly in your browser.
  </Accordion>
  <Accordion title="Work locally with the CLI">
    <Steps>
      <Step title="Install the Mintlify CLI">
        You must have Node.js version 19 or higher installed.
        
        ```bash
        npm i -g mint
        ```
      </Step>
      <Step title="Preview your site">
        Navigate to the directory containing your `docs.json` file and run:

        ```bash
        mint dev
        ```
      </Step>
    </Steps>
  </Accordion>
</AccordionGroup>

## Customize your site

Open `docs.json` to change your site name, colors, and navigation. See the [Global settings](https://www.mintlify.com/docs/organize/settings) documentation for more information.
