## Get running locally

<Steps>
  <Step title="Install the Mintlify CLI">
    ```bash
    npm i -g mint
    ```
  </Step>
  <Step title="Preview your docs">
    Navigate to your docs folder and run:
    ```bash
    mint dev
    ```
    Your docs will be live at `http://localhost:3000`.
  </Step>
  <Step title="Push to deploy">
    Commit and push your changes. If you've connected the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app), your docs will update automatically.
  </Step>
</Steps>

## Make it yours

Open `docs.json` to change your site name, colors, and navigation. The [configuration reference](https://mintlify.com/docs/settings) covers every option.

## Write content

Mintlify pages are MDX files — Markdown with components. Here are the ones you'll reach for most.

### Callouts

<Note>Use **Note** for general information.</Note>

<Tip>Use **Tip** for shortcuts and best practices.</Tip>

<Warning>Use **Warning** for things that could cause problems.</Warning>

```mdx
<Note>Use **Note** for general information.</Note>
<Tip>Use **Tip** for shortcuts and best practices.</Tip>
<Warning>Use **Warning** for things that could cause problems.</Warning>
```

### Code blocks

````mdx
```typescript
const greeting = "Hello, world!";
```
````

Use `CodeGroup` to show multiple languages side by side:

````mdx
<CodeGroup>
  ```typescript TypeScript
  const response = await fetch('/api/hello');
  ```
  ```python Python
  response = requests.get('/api/hello')
  ```
</CodeGroup>
````
