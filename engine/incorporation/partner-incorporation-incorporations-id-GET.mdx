---
title: "Get an incorporation"
openapi: "GET /partner/incorporation/{incorporationId}"
---

Retrieves a single incorporation accessible to your partner account.

Use this endpoint to refresh `incorporationStatus` and the latest `intakeState` for an ongoing incorporation.

`intakeState.validation.isComplete` tells you whether the current state is eligible to submit. `ready_for_review` is returned only after `POST /partner/incorporation/{incorporationId}/submit` succeeds for a currently complete incorporation.

Use [List incorporation issues](/engine/incorporation/partner-incorporation-issues-GET) to track active or resolved correction requests. Issues are not embedded in this response.

## Incorporation status

`incorporationStatus` can be one of:

| Status | Meaning |
|---|---|
| `collecting_information` | Required intake information, participant roles, or documents are still missing or invalid, or the current state is complete but has not been submitted for review. |
| `ready_for_review` | The incorporation was explicitly submitted and currently validates against the requirements DSL. |
| `in_progress` | Commenda is actively processing the incorporation. |
| `completed` | The incorporation has been completed. |
