BlockRush 提供多种编程语言的官方 SDK，让您可以轻松地将我们的 Solana 交易加速服务集成到您的应用程序中。

## 主要特性

- **简单集成**：所有交易方法（sendTransaction、submitBatch、sendBundle）都提供易于使用的 API
- **类型安全**：完整的 TypeScript 支持和 Rust 类型安全
- **连接管理**：内置连接池和超时控制
- **区域端点**：支持所有 BlockRush 区域端点
- **生产就绪**：经过实战检验并针对高性能应用进行优化

## 快速开始

选择您偏好的 SDK 开始使用：

1. **Rust 开发者**：查看 [Rust SDK 快速开始](/cn/solana/sdk/rust/quickstart)
2. **JavaScript/TypeScript 开发者**：查看 [JS/TS SDK 快速开始](/cn/solana/sdk/js/quickstart)

两个 SDK 都提供了构建小费交易并通过 BlockRush SWQoS 基础设施提交的示例。
