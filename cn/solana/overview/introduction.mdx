## 我们提供的服务

BlockRush 为高性能应用提供全面的 Solana 交易加速基础设施：

* SWQoS：专有的超低延迟优化技术，确保您的交易比传统 RPC 端点更快到达区块领导者，优化交易路由和包含。
* 全球区域端点：战略性地分布在全球主要地区的服务端点。
* 多种交易 API 和 JSON-RPC：灵活的交易提交方法，满足您的需求。
* 开发者 SDK：生产就绪的 SDK，实现无缝集成，支持 Rust 和 JS。
* 灵活的定价方案：多个层级满足不同需求（目前全部**免费**）。
