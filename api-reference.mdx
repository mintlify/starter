# Referência das APIs

Bem-vindo à documentação das APIs da Prefeitura do Rio de Janeiro. Aqui você encontrará a documentação completa de todas as APIs disponíveis para integração com os sistemas municipais.

## APIs Disponíveis

### Busca
API para busca centralizada da Prefeitura do Rio de Janeiro, permitindo consultas em múltiplas coleções de dados municipais.

### Serviço de Busca  
Serviço de busca avançado com funcionalidades específicas para o ecossistema da prefeitura.

### Subpav OSA SMS
API do sistema Subpav OSA (Operação de Serviços de Apoio) para gerenciamento de SMS e comunicações.

### eAi Agent
API que gerencia os fluxos e ferramentas dos agentes de IA da Prefeitura do Rio de Janeiro.

### Registro Municipal Integrado (RMI)
API do sistema RMI para acesso aos dados integrados do registro municipal.

### GO
API do sistema GO para gestão de processos municipais.

## Começando

Para utilizar qualquer uma das APIs:

1. **Autenticação**: Verifique os requisitos de autenticação na documentação específica de cada API
2. **Rate Limiting**: Respeite os limites de taxa definidos para cada endpoint
3. **Ambiente**: As APIs estão disponíveis no ambiente staging para testes
